## 位选
控制哪一个数码管亮
## 段选
控制哪一个LED发亮（1为高电平，0为低电平）共阴极与共阳极的段码位数共轭
## 静态显示代码
```c
#include <REGX52.H>
char a[10] = {0X3F,0X06,0X5B,0X4F,0X66,0X6D,0X7D,0X07,0X7F,0X6F};

void delay(int xms)	
{
	int i,j;
	for(i=0;i<xms;i++){
		for(j=0;j<120;j++);
	}
}

void seg()
{
	int i;
	for( i = 0;i<10;i++){
		P2 = a[i];
		delay(800);
	}
}
void main()
{
	
	while(1)
	{
		seg();
	}
}
```
