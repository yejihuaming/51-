## 嵌入式领域扫盲
### 嵌入式定义:
嵌入式是一种控制监视或者辅助机械和设备的装置，可以寄托在外界独立运行的一种计算机系统。
### 硬件组成：
处理器（CPU）、存储器（flash、SRAM）、外围电路（复位、时钟、AD/DA）
### 软件组成：
底层驱动、操作系统、应用程序
### 软件组成：
底层驱动、操作系统、应用程序
### 应用场景：
智能家居（家电、扫地机器人、温控开关）、便携设备（蓝牙耳机、智能手表、充电宝）工业制造、汽车交通、医疗健康等。
### MCU的定义：
MCU 的全称是微控制器，也常被称作单片机，是将中央处理器（CPU）、存储器（RAM/ROM/Flash）、输入输出接口（I/O 口），以及定时器 / 计数器、中断控制器、AD/DA 转换器、串口 / SP I/I2C 等通信外设集成在一块芯片上的专用微型计算机。
